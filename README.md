To deplay
---------

- Install hakyll 
`cabal install hakyll`

- compile and update
`make && make update`

To edit front pages
-------------------
Update the `.markdow` files

To add a lecture
----------------

- add `.lhs` file in `lectures`
- update `lectures.markdown`
